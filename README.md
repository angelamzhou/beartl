# THE BEAR: One Last Service

> ## ⚠️ SPOILER WARNING
> This is an unofficial, fan-made game inspired by FX's *The Bear*. Character traits, relationships, and the game's ending draw on plot points **through Season 5**. If you are not caught up, stop here.

A single-file, browser-based kitchen management roguelike in the style of *FTL: Faster Than Light*. Run a 12-day final stretch at The Bear: assign your brigade to stations, clear tickets before diners walk, manage prep and inventory, fight fires, and protect your restaurant's reputation.

**Play it here:** https://angelamzhou.github.io/beartl/

## Running locally

No build step, no dependencies — it's a single HTML file.

```
open index.html
```

or serve it with any static file server.

## What's in it

- FTL-style pause-and-command kitchen sim with a persistent floorplan
- A crew of characters with skills, traits, and stress that can boil over
- A component-based mise en place system — shared prep across dishes, dependency chains, par-cooking tradeoffs
- Procedural R&D projects, a cellar for ferments/dry-aging, and a daily provisions market
- **The Prep Lab** — an in-game explainer and solver for the morning prep problem as a two-stage stochastic program (sample average approximation, common random numbers, budget/ban sensitivity analysis)

This is a personal project, not affiliated with or endorsed by FX, Disney, or the creators of *The Bear*.
